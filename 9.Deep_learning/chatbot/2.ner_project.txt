## ğŸ” Step 3 Named Entity Recognition â€” `ner_extractor.py`

```python
import spacy

nlp = spacy.load(en_core_web_sm)

def extract_entities(text)
    doc = nlp(text)
    entities = {}
    for ent in doc.ents
        entities[ent.label_] = ent.text
    return entities
```
