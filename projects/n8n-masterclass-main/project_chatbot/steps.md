#### Here we are using docker
1. install docker desktop
2. open docker desktop
3. Go to project folder
4. run cmd 
5. run docker compose file
    - Rebuild images before starting
        - docker compose up --build
    - Stop containers
        - docker compose down 

    - Check running services
        - docker compose ps

        - docker ps
