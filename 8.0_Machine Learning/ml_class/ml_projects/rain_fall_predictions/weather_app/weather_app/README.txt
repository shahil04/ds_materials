Weather & Rainfall Prediction Streamlit App
-------------------------------------------

Files included:
- app.py
- pages/1_Today_Weather.py
- pages/2_Rainfall_Prediction.py
- pages/3_Forecast_Dashboard.py
- pages/4_Suggestions.py
- rainfall_prediction_model.pkl  (placeholder dummy model)
- requirements.txt

How to run:
1. (Optional) create virtual env
2. pip install -r requirements.txt
3. streamlit run app.py

Note: Replace the placeholder 'rainfall_prediction_model.pkl' with your real model if available.
Also update the OpenWeather API key in each file if you want to use your own key.
